# Hello World!

本页将带你创建一个可运行的 Aha 模块最小示例。需要一个已[安装配置好](../安装与使用.md)的 Aha 环境。

---

## 第一步：创建模块目录

在项目根目录下的 `modules/` 中创建一个新文件夹，命名为 `hello_world`：

```
Aha/
├── modules/
│   └── hello_world/      ← 新建这个文件夹
│       └── __init__.py    ← 新建这个文件
├── aha.py
└── ...
```

> 不创建文件夹直接在 `modules/` 下创建 `hello_world.py` 也可。

## 第二步：编写模块代码

在 `modules/hello_world/__init__.py` 中写入以下代码：

```python
from core.dispatcher import on_message
from models.api import Message

@on_message("hello")  # 忽略大小写，匹配用户发送的 "hello"
async def _(event: Message):
    await event.reply("world!")  # 带引用回复 "world!"
```

**如上五行代码，你的第一个模块已完成。**

### 发生了什么？

1. `@on_message("hello")` — 向 Aha 的[事件分发器](./订阅与发布事件.md)注册了一个回调：当收到内容完全匹配 `"hello"`（忽略大小写）的[聊天事件](../数据结构/事件对象.md#message)时触发。
2. `event: Message` — 回调通过关键字参数接收[事件对象](../数据结构/事件对象.md#message)，其中包含消息内容、发送者、平台等信息。
3. `await event.reply("world!")` — 向消息来源回复一条带引用的消息。

## 第三步：启动并测试

```sh
python aha.py
```

等待 Aha 启动完毕后，向你的机器人账号发送 `hello`，其就会回复 `world!`。

> 如果 Aha 因新增配置项而退出，请检查并调整 `config.dev.yml` 后重新启动。
