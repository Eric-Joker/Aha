# Aha

一个目标为~~跨平台~~、高性能、极为灵活的 Python 聊天机器人后端框架。

目前仍在毛坯期，**所有设计都有可能会在将来进行破坏性更改**。

## 特色

- **功能丰富**：实现了很多常见功能，如[**数据库长效存储**](./模块开发/内置轮子与最佳实践/长效数据.md)、[**定时任务**](./模块开发/内置轮子与最佳实践/计划任务.md)、[~~**权限系统**~~](./模块开发/内置轮子与最佳实践/权限系统.md) 等；还有很多独有特性，如[**统一配置系统**](./模块开发/统一配置系统.md)、[**通过 DSL 的消息匹配**](./模块开发/事件匹配表达式.md)、[**统一缓存及其管理**](./模块开发/内置轮子与最佳实践/缓存器.md)。

- **极优性能**：神金一样的性能追求，甚至连日志都是单开线程的。

- **可扩展性**：Aha 具有清晰、不耦合的项目结构及大量抽象层，可以轻松满足各种定制需求。
  
- **文档不齐全**：懒得写，以后再说。
